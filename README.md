# 📱 MobileHub

MobileHub is a **Mobile E-Commerce Web Application** built using **React.js**.  
The platform allows users to explore smartphones from multiple brands, view product details, and add products to their cart through a modern and responsive interface.

This project focuses on creating a **smooth user experience, reusable components, and modern UI design** using the latest frontend technologies.

---

## 🚀 Features

- 📱 Browse smartphones by **brand**
- 🛒 **Add to Cart** functionality
- 🖼️ **Product image sliders**
- 🎞️ Smooth **UI animations**
- 📱 Fully **Responsive Design**
- ⚡ Fast performance with **Vite**
- 🧭 Smooth navigation using **React components**

---

## 🛠️ Tech Stack

This project was built using the following technologies:

- ⚛️ **React.js**
- 📦 **Redux**
- 🎨 **Bootstrap**
- 🎬 **Framer Motion**
- 🎨 **CSS3**
- ⚡ **Vite**
- 🖌️ **Figma** (UI Design)
